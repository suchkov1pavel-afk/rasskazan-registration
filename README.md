# Регистрация «Рассказань 2026»

Статический сайт для GitHub Pages. Заявки отправляются в Google Таблицу через Google Apps Script.

## Обновление

1. В `config.js` вставьте URL опубликованного веб-приложения Apps Script, заканчивающийся на `/exec`.
2. В `google-apps-script/Code.gs` укажите ID Google Таблицы и папки Google Диска.
3. В Apps Script один раз выполните функцию `setupSheet()` — она создаст и красиво оформит единый лист `Регистрация`.
4. После изменения кода создайте новую версию веб-приложения через «Развернуть → Управление развертываниями».
5. Загрузите содержимое этой папки в корень репозитория GitHub Pages.

В Google Таблице используются столбцы: ФИО, Возраст, Номер телефона, Город, Церковь, Даты пребывания, Льгота, Тариф, Палатка, Спальник.


## Столбцы Google Таблицы

После запуска функции `setupSheet()` используется один лист «Регистрация» со столбцами:

ФИО · Возраст · Номер телефона · Город · Церковь · Даты пребывания · Льгота · Тариф · Оплата · Палатка · Спальник

В «Тариф» записывается сумма конкретного участника, а в «Оплата» — «Перевод» или «Наличными».
